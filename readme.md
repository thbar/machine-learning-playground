## How to install Octave

```
$ brew install octave
```

## Installing MOxUnit for unit testing

```
$ git clone https://github.com/MOxUnit/MOxUnit.git
$ cd MOxUnit
$ make install
$ cat ~/.octaverc
```

## How to install language support for atom

```
$ apm install language-matlab-octave
```
